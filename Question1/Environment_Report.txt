Question 1 – Linux Environment Verification

You have joined a company as a Linux support engineer. Before accessing company resources, you must verify your Linux environment and user account details.

Perform the necessary Linux commands and prepare a report containing:

1. Username and groups associated with your account

Command: whoami 

Output: ubuntu

Observations: The whoami command was used to identify the currently logged-in user. The command returned "ubuntu", confirming that the active account on the system is the ubuntu user. This helps verify the identity of the user before accessing system resources. The command executed successfully and displayed the expected username.

2. Current shell being used
Command: echo $SHELL
Output: /bin/bash
Observation: The echo $SHELL command displays the default shell assigned to the current user. The output shows that the Bash shell is being used. Bash is the standard command-line interpreter in many Linux distributions and is responsible for processing user commands.

3. Current working directory
Command: pwd
Output: /home/ubuntu
Observation: The pwd command prints the absolute path of the current working directory. The output indicates that the current workspace is the home directory of the ubuntu user. This helps verify the user's location within the Linux file system.
